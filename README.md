# Phone Scanner Attendance v1

Что внутри:
- `index.html` — телефон: сканер + поиск по Excel + вывод на экран
- `display.html` — внешний монитор
- `photos/` — папка с изображениями карточек
- `attendance.json` — отметки прихода
- `state.json` — текущее состояние экранов

Логика:
- фото лежат в `photos/001.jpg ... photos/320.jpg`
- при нажатии `Вывести на экран` гостю ставится отметка только в первый раз
- можно скачать новый Excel с колонкой `shown_at`
- можно обнулить результаты прихода перед стартом мероприятия
